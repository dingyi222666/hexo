## 编程经历

### 2015 – 2018.9

没有电脑的时候，对「手机跑电脑系统」很着迷，折腾过 Limbo、Bochs、APQ（QEMU）一类模拟器，也因此认识了 Windows / Linux 的基础命令。

### 2018.9 – 2019.1

接触到 [FusionApp](https://www.coolapk.com/apk/cn.coldsong.fusionapp)（基于 AndroLua+ 的网页转应用工具）。当时几乎没有编程基础，照着官方教程和群里的资料，做了第一个网页套壳 App——就是它把我带进了安卓开发和编程的大门。

之后大量下载群里的源码学习，说白了就是疯狂复制粘贴。

### 2019.1 – 2019.6

买了 Java 和安卓开发的书，真正随身带着看的只有一本，却被翻得很旧。一边继续 cv FusionApp 源码，一边在报错、修改、再运行里慢慢学会了 Lua 语法。

### 2019.7 – 2020.1

继续读源码、看 Java 书，对 Lua 有了更深一点的理解，也开始用 AndroLua+、AppLua 写东西。布局助手里的「线性布局」「相对布局」一开始完全不懂，于是一点点补 Android 基础，并用 AIDE 写 Java / Android 项目。

### 2020.1 – 2020.8

可以说是真正开始系统学习 Android 和 Lua 的阶段。Java 入门书反复看了好几遍。疫情期间基本在家，参与过青芒、氢知乎的开发，也用 Android Studio 写出了第一个项目 [light-notice](https://gitee.com/dingyi222666/light-notice)。

### 2020.8 – 2021.9

有了自己的笔记本电脑，立刻开始学 Kotlin。继续深入 Android / Java / Kotlin，也在研究 Lua 元表一类更底层的东西。

对 AndroLua 相关的代码攻防产生兴趣，写过「抽代码到 dex」一类工具并开源。`MyLuaApp` 也是在 2021 年寒假前后开始提交到 GitHub 的——这段时间更多是在实战里学。

### 2021.9 – 2021.12

参加比赛，顺带入门了 Python。

### 2022.4 – 2022.5

因为玩 QQ 机器人和原神，fork 了 [Yunzai-bot](https://github.com/Le-niao/Yunzai-Bot) 做二次开发，慢慢接触到 JavaScript。

### 2023.4 – 现在

开始做现在的主线项目 [ChatLuna](https://github.com/ChatLunaLab/chatluna)。写项目的过程中系统学了 TypeScript、Node.js 等前端相关技术。

最近一年，也花很多时间在 AI Coding 工作流上。相关思考写在：

- [AI Coding 一年来的感悟](/article/ai-coding-one-year-reflections/)
- [2026 上半年，回望我的 Vibe Coding 工作流](/article/vibe-coding-workflow-2026-h1/)
